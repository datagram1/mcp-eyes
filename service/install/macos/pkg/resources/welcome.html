Welcome to ScreenControl

Version 1.0.0

This installer will set up ScreenControl on your Mac, enabling secure remote control capabilities for AI agents and automation.

What will be installed:

  • ScreenControl Service - A background service that handles secure operations like file access, shell commands, and machine control.
  • ScreenControl Agent - A menu bar application that handles GUI operations like screenshots, clicks, and typing.

System Requirements:

  • macOS 12.0 (Monterey) or later
  • Administrator privileges for installation
  • Accessibility permissions (will be requested after installation)
